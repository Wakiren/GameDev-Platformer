# GameDev-Platformer

Controls:
- A/D to WALK
- A/D + LSHIFT to RUN
- LCTRL to DASH (while having A/D pressed)
- SPACE to JUMP/ Press again on air to DOUBLE JUMP

Debug:
- F9 to show and hide coliders
- F11 to cap/uncap FPS to 32

Notes: 
- Level 2 is done but not yet implemented into the game itself.
